建筑

单位

  • 按军种分类
    • 陆军：步兵和战车
    • 空军：飞机
    • 海军：舰船
  • 按功能分类
    • 兵营：步兵、防空、反甲、侦查、工兵、渗透、特种兵
    • 车场：机枪、防空、火炮、重炮、远攻、特种车
    • 船厂：舰艇、防空、潜艇、战舰、远攻
    • 机场：对地、对空、轰炸
    • 防御：机枪塔、防空塔、重炮塔
    • 运输：运输车、运输机
  • 按技巧分类
    • 袭扰
    • 渗透
    • 瘫痪

单位的替代

  • 苏联
    • 苏联没有渗透单位 可用恐怖机器人 模拟
  • 盟军
    • 盟军没有对地单位 可用冰冻直升机+其他攻击单位 模拟（起义时刻可以用先锋武装机）
  • 日本
    • 日本没有防空步兵 可用防空塔 天狗 打击者 火箭天使 代替（起义时刻可以用弓箭少女）

苏联

起义时刻新增单位：火炮机车、化学部队、收割机甲、粉碎者

  • 兵营
    • 步兵：动员兵
    • 防空：防空步兵
    • 反甲：防空步兵（切换技能后反甲能力更强）
    • 侦查：战熊
    • 工兵：战斗工兵
    • 渗透：无（可用恐怖机器人 模拟）
    • 特种兵：磁暴步兵（反甲）、火炮机车（反步兵）、化学部队（反步兵和反甲）
    • 英雄单位：娜塔莎
  • 车场
    • 机枪：镰刀机甲
    • 防空：牛蛙
    • 火炮：铁锤坦克
    • 重炮：天启坦克
    • 远攻：V4导弹发射车
    • 特种车：收割机甲（对地和防空）、粉碎者（瘫痪）
  • 船厂
    • 舰艇：磁暴快艇
    • 防空：牛蛙
    • 潜艇：阿库拉潜艇
    • 战舰：无（可用阿库拉潜艇模拟）
    • 远攻：无畏战列舰
  • 机场
    • 对地：双刃直升机
    • 对空：米格战斗机
    • 轰炸：基洛夫飞艇
  • 防御
    • 机枪塔：哨兵枪
    • 防空塔：防空炮
    • 重炮塔：磁暴线圈
  • 运输
    • 运输车：牛蛙
    • 运输机：双刃直升机
  • 袭扰：恐怖机器人、双刃直升机、基洛夫飞艇
  • 渗透：无（可用恐怖机器人 模拟）
  • 瘫痪：战熊、磁暴步兵、恐怖机器人、粉碎者

盟军

起义时刻新增单位：冰冻军团、平定者、未来坦克、先锋武装机

  • 兵营
    • 步兵：维和步兵
    • 防空：标枪兵
    • 反甲：标枪兵
    • 侦查：警犬
    • 工兵：工程师
    • 渗透：间谍
    • 特种兵：冰冻军团（瘫痪）
    • 英雄单位：谭雅
  • 车场
    • 机枪：激流快艇（ACV）
    • 防空：多功能战车（IFV）
    • 火炮：守护者坦克
    • 重炮：幻影坦克
    • 远攻：雅典娜炮
    • 特种车：平定者（重炮）、未来坦克（重炮）
  • 船厂
    • 舰艇：激流快艇（ACV）
    • 防空：水翼船
    • 潜艇：无（可用海豚模拟）
    • 战舰：突袭驱逐舰
    • 远攻：航母
  • 机场
    • 对地：无（可用冰冻直升机+其他攻击单位 模拟（起义时刻可以用先锋武装机））
    • 对空：阿波罗战斗机
    • 轰炸：维和轰炸机、世纪轰炸机、先锋武装机
  • 防御
    • 机枪塔：多功能炮塔 + 维和步兵
    • 防空塔：多功能炮塔 + 标枪兵
    • 重炮塔：光棱塔
  • 运输
    • 运输车：激流快艇（ACV）、多功能战车（IFV）
    • 运输机：世纪轰炸机
  • 袭扰：维和轰炸机、世纪轰炸机、冰冻直升机、先锋武装机
  • 渗透：间谍
  • 瘫痪：警犬、冰冻军团、冰冻直升机、水翼船、航母

经验

战斗的类型

  • 陆地战：陆军 + 对地 + 防空
  • 登陆战：海军 + 对地 + 防空

战斗的经验

  • 机枪反步兵
  • 重炮反战车
  • 远攻反防守

如何防止渗透单位

  • 用围墙将自己保护起来
  • 用侦查单位让渗透单位现出原形

如何让防守的单位固定不动

  • 建造围墙将单位围起来
  • 进入建筑后可以固定不动
  • 苏联的工兵可以建造堡垒
  • 苏联的收割者可以固定不动
  • 盟军的平定者可以固定不动
  • 日本的防御塔可以固定不动

如何进行登录作战

  • 使用运输车
  • 使用运输机
  • 使用两栖单位

快捷键

  • 选择
    • left click 选择（按住shift时是加入操作）
    • right click 选择并执行默认操作
      • 空地 移动
      • 建筑 进驻
      • 敌人 攻击
    • 选择所有单位 Q
    • 选择同类单位 W 选择视野内的同类单位（按多次可以扩大范围）
    • 直接选择子项 f1~f12 直接切换焦点到子项（按多次可以选择某个子项）
    • 循环选择子项 tab/shift + tab 循环切换焦点到子项（不是选择某个子项）
  • 攻击
    • 常规攻击 right click 选择某个目标进行攻击
    • 行进攻击 A 行进到指定地点（移动的过程中遇到敌人会进行攻击）
    • 停止攻击 S
    • 强制攻击 ctrl + right click 可以攻击空地、建筑和桥梁
  • 移动
    • 常规移动 right click 选择某片空地进行移动
    • 停止移动 S
    • 倒退移动 D 可以省去掉头时间
    • 强制移动 G 可以实现碾压效果
  • 工厂
    • 基地 E 生产各种建筑（按多次会在多个基地的建筑中切换）
    • 防御 R 生产防御武器（按多次会在多个基地的建筑中切换）
    • 兵营 T 生产各种步兵（按多次会在多个基地的建筑中切换）
    • 车场 Y 生产各种战车（按多次会在多个基地的建筑中切换）
    • 机场 U 生产各种飞机（按多次会在多个基地的建筑中切换）
    • 船厂 I 生产各种舰船（按多次会在多个基地的建筑中切换）
  • 集结
    • ctrl + r 设置集结点
    • shift + right click 设置所有单位的集结点（需要先选中某个建筑）
  • 编队
    • ctrl + num 加入某个编队
    • num 选择某个编队（按多次可以切换视角到编队处）
    • shift + num 混入某个编队（临时性的加入某个编队）
  • 视角
    • H 切换视角到基地（按多次不会在多个基地之间切换）
    • Space 切换视角到最近的事件发生点
    • ctrl + jkl 设置视角书签
    • jkl 选择视角书签
  • 模式
    • alt 路径点模式
    • ctrl + z 计划模式
    • alt + a 侵略模式 会攻击非战斗单位和建筑
    • alt + s 警戒模式 会攻击视野内的敌人，还会追击一段路程
    • alt + d 固守模式 会攻击视野内的敌人，但不会进行追击
    • alt + g 停火模式 不会攻击敌人，偷袭时可以切换到此模式
  • 其他
    • F 技能
    • N 切换到向下一个支援单位
    • X 散开
    • Z 售卖
    • C 维修
