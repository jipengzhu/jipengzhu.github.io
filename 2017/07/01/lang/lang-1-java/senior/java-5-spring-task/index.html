TaskConfig

task_config 的属性如下

  • 任务id
  • 任务名称
  • 任务描述
  • 任务超时
  • 任务分组
  • 任务优先级

TaskRecord

task_record 的属性如下

  • 记录id
  • 任务id
  • 任务参数
  • 任务结果
  • 任务状态
  • 任务创建时间
  • 任务开始时间
  • 任务结束时间
  • 任务耗时

TaskState

task_state 的属性如下

  • 已创建 10
  • 运行中 20
  • 已取消 30
  • 已中断 40
  • 已成功 50
  • 已失败 60
  • 已超时 70

ps：任务在 运行前 可以被 取消 ，在 运行时 只能被 中断

DelayedTask

  • Thread.sleep实现延时任务
  • DelayQueue实现延时任务

实现延时任务的方案

  • Thread.sleep：会导致线程暂停，效率低
  • DelayQueue和MQ：方案较复杂，不推荐
  • TaskScheduler.schedule(Runnable task, Instant startTime)：方案较简单，可推荐
