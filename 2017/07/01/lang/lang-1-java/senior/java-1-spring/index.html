AOP同一个类方法调用时为什么不使用代理对象

因为代理类的实例是在代理类生成后才能创建，
因此在代理类生成的时候进行同一个类的方法调用时代理类的实例还不存在，
所以只能通过原生类创建一个原生对象进行进行同一个类的方法。

注：在方法里面通过AopContext.currentProxy和ApplicationContext.getBean代替this调用从而可以使用代理对象
