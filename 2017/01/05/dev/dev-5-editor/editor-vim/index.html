  • 普通模式：按esc进入
  • 编辑模式：普通模式时按i（insert）、a（append）、o进入
  • 命令模式：普通模式时按冒号（: 执行命令）或斜杠（/ 向前搜索）或问号（? 向后搜索）进入
  • 可视模式：普通模式时按v（visual 会选中一个字）或V（Visual 会选中一整行）进入

Edit Mode

  • i：在光标前插入
  • I：在行首（bol）处插入
  • a：在光标后插入
  • A：在行尾（eol）处插入
  • o：在下一行插入空行
  • O：在上一行插入空行

Command Mode

  • :（冒号）：执行命令
  • /（斜杠）：向前搜索
  • ?（问号）：向后搜索

快捷键如下

  • 移动到行首：home、ctrl + b
  • 移动到行尾：end、ctrl + e
  • 移动到（上一个）词首：ctrl + arrow left
  • 移动到（下一个）词尾：ctrl + arrow right
  • 从光标前删除到行首：ctrl + u
  • 从光标前删除到词首：ctrl + w

Visual Mode

  • v：进入可视模式，并选中一个字
  • V：进入可视模式，并选中一整行

Command

  • vim之命令g详解
  • vim之执行外部命令入门
  • vim之执行外部命令详解

vim命令格式：cmd[n][space|motion]

  • x：cut，剪切，会停留在普通模式
  • s：substitute，剪切，会进入编辑模式
  • d：delete，剪切，会停留在普通模式
  • c：change，剪切，会进入编辑模式
  • r：replace，替换
  • y：yank，复制
  • p：paste，粘贴，p（粘贴到光标后）、P（粘贴到光标前）
  • g：goto，跳转

Motion

  • vim之motion入门
  • vim之motion进阶
  • vim之motion详解

vim移动格式：[n]motion

  • h：左移
  • l：右移
  • j：下移
  • k：上移
  • b：移动到（上一个）词首
  • e：移动到（下一个）词尾
  • w：移动到下一个词首
  • 0：移到行首
  • ^：移到行首（第一个非空白字符）
  • $：移到行尾
  • gg：移到页顶
  • G：移到页底

  • x：保存退出（有修改时）或者退出（无修改时）
  • w!：强制保存不退出
  • q!：强制不保存退出
  • wq!：强制保存退出

vim替换格式：[range]s/source/target/option

range

range的边界格式如下

  • .：代表当前行
  • n：代表第n行
  • %：代表所有行
  • $：代表最后一行
  • /search-pattern/：代表下一个搜索结果所在的行
  • ?backwards-search-pattern?：代表上一个搜索结果所在的行

ps：区间分隔符用逗号（, ）或者分号（; ）都可以的

range是左闭右闭的，省去range使用光标所在的行, range的一边缺省使用光标所在的行代替那一边

  • %：所有行
  • 1, 3：第1到3行
  • 3, $：第3行到最后一行
  • 1, .：第1行到当前行
  • ., $：当前行到最后一行
  • 1, ：第1行到当前行
  • , $：当前行到最后一行

option

  • g：表示全局替换，将匹配到的所有项都替换掉，而不仅仅是第一个
  • c：表示每次替换前需要确认，即弹出一个界面供用户确认是否替换
  • e：表示没有匹配项时不显示错误
  • i：表示忽略大小写
  • I：表示区分大小写
