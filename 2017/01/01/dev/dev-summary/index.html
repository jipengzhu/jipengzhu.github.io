  • Grammar（文法）、Syntax（语法）、Language（语言）

  • Transaction（事务）、Durability（持久化）
  • Update（更新）、Change（变更）、Modify（修改）

Status

  • Status和State的区别
  • Status：状态，状态之间有依赖关系
  • State：状态，状态之无依赖关系

Parameter

  • Parameter和Argument的区别
  • Parameter：定义声明的参数
  • Argument：实际传入的参数

Util

  • Util和Tool的区别
  • Util：功用
  • Tool：工具

Service

  • Service、Manager和Helper
  • Service：服务类
  • Manager：管理类
  • Helper：辅助类

Inheritance

  • 继承（Inheritance）、组合（Composition）、聚合（Aggregation）技术详解
  • 混入（Mixin）技术详解
  • 继承（Inheritance）和混入（Mixin）对比
  • Inheritance：继承
  • Composition：组合（强依赖）
  • Aggregation：聚合（弱依赖）
  • Mixin：混入

Framework

  • 组件（component）技术详解
  • 中间件（Middleware）技术详解
  • Framework：框架，被开发者继承
  • Library：库，被开发者组合（强依赖）
  • Component：组件，被开发者聚合（弱依赖）
  • Middleware：中间件，被开发者使用

Plugin

  • Plugin和Extend的区别
  • Plugin：插件，新增没有的功能
  • Extend：扩展，增强已有的功能

Find

  • Find：查找，查找文本
  • Query：查询，查询结果
  • Search：搜索，搜索一切

Availability

  • Availability和Reliability的区别
  • Availability：可用性（关注的是性能）
  • Reliability：可靠性（关注的是质量）

Program

  • Program：程序，指令和数据的合集，支持打包和部署，可以启动和停止
  • Process：进程，程序（program）的运行态（runtime）
  • Software：软件，程序（program）的封装（package），支持安装和卸载，可以启动和停止，还可以开机自启
  • Service：服务，软件（software）的运行态（runtime）
  • Application：应用，功能（function）的合集

Program是操作系统的视角，Application是开发者和使用者的视角
