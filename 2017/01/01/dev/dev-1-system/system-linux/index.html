Process

Basic

Program

程序（program）：代码的合集，支持分发和部署，可以启动运行和停止运行
进程（process）：程序（program）的运行态（runtime）

Software

软件（software）：程序的封装，支持安装和卸载，可以启动运行和停止运行，还可以开机自启动
服务（service）：软件（software）的运行态（runtime）

Application

应用（application）：抽象时指功能的合集，具体时指远程服务使用的程序或者手机客户端使用的程序
