data

Data

文件：File
数据库：Database
信息：Info

ps：数据库的存储部分本质上还是文件

File

  1. 文件分为文本文件（Text File）和 二进制文件（Binary File）
  2. 文本文件又分为普通文本文件（Plain Text）和富文本文件（Rich Text）
  3. 文件的格式包含编码格式（File Encoding）和数据格式（File Format）

文本文件 俗称 文档 ， 二进制文件 俗称 文件

ps：口语中的文件格式或者文件类型（File Type）一般指文件的数据格式

Text File

Encoding

ascii ansi unicode utf-8 gbk gb2312 gb18030

Format

文档类的有

  • txt
  • doc、docx
  • ppt
  • excel
  • pdf

标记类的有

  • html
  • xml
  • md

程序类的有

  • java
  • python
  • js

配置类的有

  • properties
  • ini
  • xml
  • json
  • yaml
  • toml

思维类的有

  • xmap（Xmind）

绘图类的有

  • psd（Photoshop）
  • vsd、vsdx（Visio）
  • drawio（DrawIO）

设计类的有

  • rp（Axure）
  • sketch（Sketch）
  • figma（Figma）

分类非常多，还有其他各种软件所生成的文件

Binary File

Encoding

  • ascii

ps：二进制文件只有一种编码格式，即ascii编码

Format

  • 图像：Image
  • 音频：Audio
  • 视频：Video

图像又主要分为3类

  • 相片：Photograph（照相机拍摄产生的图像）
  • 截图：Screenshot（屏幕截取产生的图像）
  • 图片：Picture（画图软件生成的图像）

ps：图形学（Graphics）指的是图像（Image）的绘制（Draw）和编辑（Edit）技术

Info

信息传输分类 有线 无线
信息传输载体 光缆（电缆） 基站 卫星
信息传输方式 点对点（P2P：peer to peer） 广播（broadcast）
信息传输格式 短信 邮件 消息（通讯类软件）
信息传播方式 新闻（官方传播形式） 媒体（民间传播形式）
信息传播格式 文字（文学） 图像（美术） 音频（音乐） 视频 表演

ps：传输是通信学的术语，传播是语言学的术语

文字类型分类 文章 帖子
图像类型分类 图片 画作
音乐类型分类 歌唱 器乐
视频类型分类 电影 电视剧
表演类型分类 舞蹈 舞台剧

ps：文章组织形式 随笔 专题
