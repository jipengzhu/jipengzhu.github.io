俗称为 链表 （link）

  • 元素（element）：包含各种数据的对象
  • 结点（node）：包含元素（element）和指向下一个结点（node）的指针（java中为引用）
  • 头结点：链表中的第一个结点（包含的元素一般为空而不是第一个元素）
  • 尾结点：链表中的最后一个结点
  • 元结点：链表中包含第一个元素的结点（当头结点包含第一个元素时头结点等于元结点）
  • 头指针：指向链表的头部（head），即指向头结点
  • 尾指针：指向链表的尾部（tail），即指向尾结点

头结点的作用

  • 简化操作：无头结点时添加和删除元结点需要特殊处理（更新head指针），有头结点时不需要

逻辑结构

  • 可选（optional）：包含一个元素的容器（container）结构
  • 合集（collection）：包含多个元素的容器（container）结构
  • 集合（set）：无序的合集（collection）
  • 列表（list）：有序的合集（collection）
    • 队列（queue）：只支持尾部插入和头部删除的列表（list）
    • 栈列（stack）：只支持头部插入和头部删除的列表（list）
  • 字典（dict）：支持键和值快速关联的映射（mapping）结构
  • 堆（heap）：父结点比子结点大或小（兄弟结点无大小关系）的树形（tree）结构
  • 跳表（skiplist）：多层链表组合的形成的表格（table）结构

优先级队列实现：堆
可排序字典实现：红黑树或者跳表

编号关系证明如下

  • 记完全二叉树中当前节点的编号为t（t>=1），层级为k（k>=1）
  • 因为第k层最左边的元素的编号等于第1层到第k-1层的所有元素个数之和再加1
  • 所以第k层最左边的元素的编号为2^0 + … + 2^(k-2) + 1 = 2^(k-1)
  • 所以第k+1层最左边的元素的编号为 2^(k+1-1) = 2^k
  • 因为t节点的左孩子编号等于第k+1层最左边的元素的编号加上t节点左边节点的孩子个数和
  • 所以t节点的左孩子编号为2^k + 2*(t-2^(k-1)) = 2 * t
  • 因为t节点的右孩子编号等于做左孩子的编号加1
  • 所以t节点的右孩子编号为2 * t + 1
  • 由左孩子和右孩子编号公式可知，左孩子是偶数，右孩子是奇数
  • 假设t节点为左孩子节点
  • 因为左孩子节点是父节点的2倍，所以t节点的父节点为t / 2 = t的二分之一并取整（此时t为偶数）
  • 假设t节点为右孩子节点
  • 因为左孩子节点是父节点的2倍，所以t节点的父节点为(t - 1) / 2 = t的二分之一并取整（此时t为奇数）
  • 综上t节点的左孩子编号为2 * t，右孩子编号为2 * t + 1，父节点的编号为t / 2（取整除法）

下标关系证明如下

  • 记完全二叉树中当前节点在数组中的下标为i（i>=0，和t相差1）
  • 由编号公式可知左孩子下标为(i + 1) * 2 - 1 = 2 * i + 1
  • 由编号公式可知右孩子下标为(2 * i + 1) + 1 = 2 * i + 2
  • 由编号公式可知父节点下标为(i + 1) / 2 - 1 = (i - 1) / 2
  • 综上左孩子下标为2 * i + 1，右孩子下标为2 * i + 2，父节点下标为(i - 1) / 2（取整除法）

ps：下标转编号时需要加1，编号转下标时需要减1

  • 插入排序：将未排序部分里的元素挨个和已排序的元素比较并插入到已排序部分里的合适位置上

    • 希尔排序
  • 选择排序：在未排序部分挨个和最小的元素比较并选择从而将最小的元素放到未排序部分的首位
